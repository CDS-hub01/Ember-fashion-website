# EMBER — Luxury Fashion E-Commerce Website

A multi-page luxury fashion e-commerce website built from scratch with pure HTML, CSS and JavaScript — no frameworks, no libraries.

## Overview

EMBER is a fictional high-end fashion brand website featuring a full shopping experience: browsing collections, filtering and sorting products, viewing detailed product pages, and managing a persistent shopping cart — all powered by vanilla JavaScript.

## Features

- Dynamic product rendering from a JavaScript data array
- Category filtering and price sorting on the shop page
- Individual product detail pages with image gallery, size selection and quantity controls
- Shopping cart with add, remove and quantity updates using `localStorage`
- Newsletter signup with email validation
- Responsive hamburger navigation
- Fully responsive design across mobile, tablet and desktop
- Custom dark charcoal aesthetic with editorial-style typography

## Built With

- HTML5
- CSS3 (custom properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)

## Pages

- `Ember.html` — Home page with featured products
- `shop.html` — Full collection with filters and sorting
- `product.html` — Dynamic product detail page
- `cart.html` — Shopping cart and checkout summary
- `lookbook.html` — Editorial imagery
- `about.html` — Brand story

## Key JavaScript Concepts Used

- Array and object data structures
- `filter()`, `sort()`, `map()`, `find()`, `reduce()`
- DOM manipulation and template literals
- `localStorage` for persistent cart state
- URL parameters for dynamic product pages
- Event delegation and form validation
